Key Inspirations

The interactive story “The Boat” from SBS was the driving force of my inspiration for this project. The original text was embellished with vivid illustrations, rich sounds, and effective motions. All the elements combined to create an immersive narrative and atmosphere. The story was compelling.
John Ashbery is a key inspiration for my poetic practice. His poetry holds many writing techniques, playing with the written word. He has written both long, stream-of-conscious type poems and shorter, to-the-point poems. At times, there is a certain feeling of dialogue within his work – like he is talking to the reader, giving a monologue, sometimes having a moment where his thoughts are interrupted. His work highlights everyday poetic details and delivers soulful feelings. I feel a sense of connection and authenticity when reading his work.

Resources

This project utilizes many resources to achieve its completion, including GASP, Animate.css, and YouTube videos. GSAP from GreenSock facilitated the animation process; this widely-used JavaScript animation library is easy to learn with simple lines of code and detailed documentation. Animate.css is a quick-to-use library which takes away the manual labour of css animations. Its documentation is clear and concise, helping you start animating aspects of your webpage in a jiffy. YouTube is a great gateway to information. There, I found Coding Snow who taught how to reveal website aspects when scrolling past a relative point with their video “Reveal Website Elements On Scroll | On Page Scroll Down - Using HTML, CSS & Javascript.” Also, I learnt how to do css animations with code from Easy Tutorials’ video “How To Make Animated Website Design Using HTML And CSS Step By Step Tutorial 2020.” The internet provides access to many great resources – I’m grateful for their help in creating my project.
